## ✨ 스터디 소개
본 스터디는 컴퓨터 공학 핵심 이론(CS)을 학습하고, 실무 프로젝트 경험과 연결하여 면접 대비와 실전 역량을 함께 키우는 것을 목표로 합니다.

- ✅ CS 이론 발표 + 프로젝트 경험
- ✅ GitHub를 통한 자료 정리
